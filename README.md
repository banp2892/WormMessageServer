# WormMessageServer
## Что это?
TCP-сервер на c++ для Windows для приема данных от червя. Используется библиотека ws2_32.lib
## Как это работает?
1. **Подготовка:** Сервер инициализирует Winsock и биндится (bind) на указанный порт (например, `8080`).
2. **Ожидание:** При вызове `start()` сервер переходит в режим `listen` и блокируется до появления клиента (`accept`).
3. **Прием данных:** После подключения сервер считывает данные из буфера и выводит их в консоль.

## Пример использования
```cpp
TcpServer server(8080);
if (server.start()) {
    std::string data = server.receiveData();
    std::cout << "Data: " << data << std::endl;
}
